Color Cascade — це захоплююча гра на реакцію, де гравець керує платформою для збору крапель різного кольору. Гра проста, але динамічна, і вимагає швидких рефлексів та уважності.

Особливості гри:
1. Захоплюючий геймплей: Збирайте кольорові краплі для отримання очок.
2. Різні типи крапель: Золоті краплі приносять більше очок, радужні додають життя, а чорні можуть зменшувати життя, якщо їх зібрати.
3. Система паузи: Зупиніть гру будь-коли за допомогою кнопки "Пауза".
4. Таблиця рекордів: Гра автоматично зберігає найвищий рахунок.

Як грати:
Використовуйте клавіші стрілка вліво та стрілка вправо для переміщення платформи.
Збирайте кольорові краплі, уникаючи пропусків.
Не збирайте чорні краплі — вони зменшують кількість життів.
Натискайте кнопку "Пауза", щоб зупинити або відновити гру.

Типи крапель:
Червоні, сині, жовті: приносять 1 очко.
Золоті: приносять 5 очок.
Радужні: додають 1 життя.
Чорні: зменшують життя, якщо зібрати.

Інструкція для запуску:
Завантажте всі файли проєкту у одну папку:
index.html
styles.css
game.js
Відкрийте файл index.html у будь-якому сучасному браузері (наприклад, Google Chrome, Firefox, Microsoft Edge).
Гра автоматично розпочнеться. Натискайте стрілки для переміщення платформи і починайте збирати краплі!
